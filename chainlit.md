# Pile SM

Local Scrum Master assistant. Bilingual: Vietnamese & English.

**Quick start:** Click a button below or type your question.

**Capabilities:**
- Sprint tracking, standup, workload, blockers, cycle time
- Jira: search, create/update issues, manage sprints, backlog, epics
- Memory: remember decisions, forget, search past context
- Knowledge: upload PDF/markdown for methodology advice
- Browser: scrape web pages when API unavailable
- Charts: auto-rendered when response contains numeric data
